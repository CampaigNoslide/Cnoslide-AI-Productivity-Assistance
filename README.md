# Cnoslide – AI Productivity Assistant

Atlas is a full SaaS-style AI workplace assistant built with Lovable AI. It provides multiple productivity tools in one unified dashboard, including AI chat, email generation, meeting summarisation, task planning, and research assistance.

---

## 🚀 Features

### 📊 Dashboard
- Productivity overview with metrics
- Quick-access tool cards
- Clean SaaS-style layout

### 💬 AI Chat Assistant
- Streaming AI responses
- Conversation thread history (localStorage)
- Suggestion chips for quick prompts
- Fully responsive chat interface

### ✉️ Smart Email Generator
- 4 tone options (formal, casual, professional, friendly)
- Editable output
- Copy & regenerate support

### 📝 Meeting Summariser
- Converts notes into structured Markdown
- Key points, decisions, action items, deadlines

### 📅 Task Planner
- Daily and weekly planning
- Time-blocked suggestions
- Productivity tips

### 🔍 Research Assistant
- TL;DR summaries
- Key insights and trends
- Recommendations

---

## 🎨 UI / UX
- Modern SaaS design system (blue, white, dark grey)
- Light & Dark mode toggle
- Fully responsive sidebar layout
- Clean chat-first interface
- AI disclaimer footer

---

## ⚙️ Tech Stack
- React / Next.js (Lovable generated)
- Tailwind CSS (styling)
- Gemini AI (google/gemini-3-flash-preview)
- REST API (/api/chat streaming)
- LocalStorage for chat history
